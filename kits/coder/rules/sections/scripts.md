## 📜 SCRIPTS (Automation)

**When to run:**
- Before PR/commit → `checklist.py` (Security, Lint, Tests)
- Before deploy → `verify_all.py`
- Security changes → `security_scan.py`
- DB changes → `schema_validator.py`
- API changes → `api_validator.py`
- UI changes → `a11y_checker.py`
